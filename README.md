## 初期キーマップ
### Tap Dance
|name|tap|hold|2tap|tap hold|
|:-:|:-:|:-:|:-:|:-:|
|tp1|Tab|Ctrl|Q||
|tp2||Shift||Ctrl|
|tp3|半／全|layer to_ALT|||
|tp4|F7|layer to_ALT|F8||
